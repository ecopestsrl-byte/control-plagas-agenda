CONTROL PLAGAS - VERSION PARA GITHUB Y RENDER

Sube el contenido de esta carpeta a GitHub.

En GitHub debe quedar asi:

package.json
server.js
render.yaml
README.md
GUIA-SINCRONIZACION-NUBE.txt
public/
data/

No subas el ZIP completo a Render.
Primero descomprime y sube los archivos.

En Render:

Build Command: npm install
Start Command: npm start

Para que los datos no se pierdan, agrega Persistent Disk:

Mount Path:
/opt/render/project/src/data

admin
