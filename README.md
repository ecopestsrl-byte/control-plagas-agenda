# Control Plagas - Agenda en la nube

Version preparada para usar desde varias computadoras con el mismo enlace.

## Inicio local

```bash
npm install
npm start
```

Abre:

```text
http://localhost:8080
```

## Acceso inicial

- Usuario: `admin`
- Contrasena: `admin123`
- Codigo de Compania: `CP`

## Sincronizacion

Para sincronizar casa y trabajo, todos deben entrar al mismo servidor publico. No uses copias separadas en cada computadora.

En Render u otro hosting, usa un disco persistente para la carpeta `data`.
